# Projet Sio Crawler #

présentation du projet : le but est de réaliser un jeu de rôle permettant un jeu peu massivement
multi joueur permettant aux étudiants du bts sio de se divertir pendant les heures de cantine
mais surtout d'améliorer leurs compétences en développement.

Les outils mis en oeuvre :

* Git
* Visual Studio
* MySql
* Apache

Le développement tourne autour de 3 grandes parties



1. L'inscription en ligne
2. Développement du jeu en lui même permettant l'exploration d'un labyrinthe
3. La sauvegarde des personnages et du contexte du jeu





| Developpement          | Langage     | Technique deprogrammation                                |
| ---------------------- | :---------: | :------------------------------------------------------: |
| Inscription en ligne   | PHP, MySql  | développement web MVC avec Code Igniter                  |
| Sio crawler le jeu     | C#		   | programmation objet, tests unitaires                     |
| Sauvegarde du contexte | C#, MySql   | programmation procédural procédures stockées en mysql    |
